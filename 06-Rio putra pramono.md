<h2>📱 Web & Mobile Computing (WMC)</h2>

<p>
WMC adalah konsep pengembangan aplikasi yang dapat diakses melalui
web maupun perangkat mobile sehingga pengguna dapat mengakses sistem
kapan saja dan di mana saja menggunakan internet.
</p>

<ul>
<li>Multi Platform</li>
<li>Realtime System</li>
<li>Berbasis Internet & Cloud</li>
</ul>
</div>

<!-- MOBILE APP -->
<div class="card">
<h2>⚙️ Teknologi Mobile Application</h2>

<h3>1. Native Application</h3>
<p>Aplikasi dibuat khusus untuk Android atau iOS dengan performa tinggi
dan akses hardware penuh.</p>

<h3>2. Hybrid Application</h3>
<p>Aplikasi gabungan web dan mobile menggunakan satu kode untuk banyak platform.</p>

<h3>3. Progressive Web App (PWA)</h3>
<p>Website yang dapat di-install seperti aplikasi mobile dan mendukung offline mode.</p>

</div>

<!-- PREP -->
<div class="card">
<h2>🧠 Framework PREP</h2>

<p>
Framework PREP digunakan untuk menyampaikan ide atau solusi teknologi
secara sistematis dan mudah dipahami.
</p>

<div class="prep">
<h3>P — Point</h3>
<p>PWA merupakan solusi efektif dalam pengembangan aplikasi WMC modern.</p>
</div>

<div class="prep">
<h3>R — Reason</h3>
<p>
Karena PWA dapat berjalan di berbagai perangkat tanpa perlu membuat
aplikasi Android dan iOS secara terpisah.
</p>
</div>

<div class="prep">
<h3>E — Example</h3>
<p>
Contohnya aplikasi katalog produk, sistem absensi online,
e-commerce, dan aplikasi kampus berbasis web mobile.
</p>
</div>

<div class="prep">
<h3>P — Point (Reinforce)</h3>
<p>
Dengan demikian PWA menjadi teknologi yang efisien,
cepat dikembangkan, dan cocok digunakan pada konsep WMC.
</p>
</div>

</div>

terstruktur, dan mudah dipahami.
