Web & Mobile Computing (WMC)
Teknologi Mobile App dan Framework PREP

📱 Web & Mobile Computing (WMC)
WMC adalah konsep pengembangan aplikasi yang dapat diakses melalui web maupun perangkat mobile sehingga pengguna dapat mengakses sistem kapan saja dan di mana saja menggunakan internet.

Multi Platform
Realtime System
Berbasis Internet & Cloud
⚙️ Teknologi Mobile Application
1. Native Application
Aplikasi dibuat khusus untuk Android atau iOS dengan performa tinggi dan akses hardware penuh.

2. Hybrid Application
Aplikasi gabungan web dan mobile menggunakan satu kode untuk banyak platform.

3. Progressive Web App (PWA)
Website yang dapat di-install seperti aplikasi mobile dan mendukung offline mode.

🧠 Framework PREP
Framework PREP digunakan untuk menyampaikan ide atau solusi teknologi secara sistematis dan mudah dipahami.

P — Point
PWA merupakan solusi efektif dalam pengembangan aplikasi WMC modern.

R — Reason
Karena PWA dapat berjalan di berbagai perangkat tanpa perlu membuat aplikasi Android dan iOS secara terpisah.

E — Example
Contohnya aplikasi katalog produk, sistem absensi online, e-commerce, dan aplikasi kampus berbasis web mobile.

P — Point (Reinforce)
Dengan demikian PWA menjadi teknologi yang efisien, cepat dikembangkan, dan cocok digunakan pada konsep WMC.

⭐ Kesimpulan
WMC menggabungkan teknologi web dan mobile untuk menghasilkan aplikasi modern. Teknologi mobile terdiri dari Native, Hybrid, dan PWA. Framework PREP membantu menjelaskan solusi teknologi secara logis, terstruktur, dan mudah dipahami.
