<h2>📱 Web & Mobile Computing (WMC)</h2>

<p>
Web & Mobile Computing (WMC) merupakan konsep pengembangan sistem
informasi yang memanfaatkan teknologi web dan perangkat mobile
untuk memberikan akses layanan secara fleksibel, cepat, dan real-time.
Melalui WMC, pengguna dapat mengakses aplikasi kapan saja dan di mana saja
menggunakan jaringan internet.
</p>

<p>
Konsep ini berkembang karena meningkatnya penggunaan smartphone,
cloud computing, dan kebutuhan sistem digital yang dapat berjalan
di berbagai perangkat tanpa batasan platform.
</p>

<ul>
<li>Mendukung multi-device (desktop dan mobile)</li>
<li>Menggunakan konektivitas internet</li>
<li>Menyediakan layanan real-time</li>
<li>Meningkatkan mobilitas pengguna</li>
</ul>
</div>

<div class="card">
<h2>⚙️ Teknologi Mobile Application</h2>

<h3>1. Native Application</h3>
<p>
Native Application adalah aplikasi yang dikembangkan khusus untuk satu
sistem operasi tertentu seperti Android atau iOS. Aplikasi jenis ini
memiliki performa tinggi karena langsung berinteraksi dengan sistem
operasi dan perangkat keras seperti kamera, GPS, dan sensor perangkat.
</p>

<h3>2. Hybrid Application</h3>
<p>
Hybrid Application merupakan kombinasi antara teknologi web dan aplikasi
mobile. Pengembang cukup membuat satu basis kode menggunakan HTML,
CSS, dan JavaScript yang kemudian dapat dijalankan di berbagai platform.
Pendekatan ini menghemat waktu pengembangan dan biaya produksi aplikasi.
</p>

<h3>3. Progressive Web App (PWA)</h3>
<p>
Progressive Web App adalah aplikasi berbasis web yang memiliki pengalaman
pengguna seperti aplikasi mobile. PWA dapat di-install pada perangkat,
mendukung mode offline, serta memiliki performa ringan tanpa harus
melalui proses instalasi dari app store.
</p>

<p>
PWA menjadi solusi populer dalam konsep WMC karena mampu menjembatani
kebutuhan aplikasi web dan mobile secara bersamaan.
</p>

</div>

<h2>🧠 Framework PREP</h2>

<p>
Framework PREP adalah metode penyampaian ide atau argumentasi yang
digunakan dalam analisis teknologi maupun presentasi akademik.
Metode ini membantu penyampaian informasi menjadi lebih terstruktur,
jelas, dan mudah dipahami.
</p>

<div class="prep">
<h3>P — Point</h3>
<p>
Point merupakan pernyataan utama atau ide pokok yang ingin disampaikan.
Pada konteks WMC, ide utamanya adalah bahwa PWA menjadi solusi modern
dalam pengembangan aplikasi mobile.
</p>
</div>

<div class="prep">
<h3>R — Reason</h3>

Reason menjelaskan alasan mengapa ide tersebut benar. PWA dianggap

mengembangkan aplikasi secara terpisah.
</p>
</div>

digital yang dapat diakses melalui web maupun mobile.
</p>
