# Kasir PWA

Aplikasi kasir sederhana berbasis web (PWA)

## Fitur
- Tambah produk ke keranjang
- Hitung total otomatis
- Bisa digunakan offline
- Notifikasi online / offline

## Cara Jalankan
Gunakan Live Server / Docker / localhost
